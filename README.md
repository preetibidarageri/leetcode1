# Two Sum Problem (LeetCode)

## Problem
Given an array of integers `nums` and an integer `target`, return indices of two numbers such that they add up to target.

## Approach
- Use a HashMap (dictionary)
- Store elements as we iterate
- Check if the required difference exists

## Example
Input:
nums = [2,7,11,15], target = 9

Output:
[0,1]
